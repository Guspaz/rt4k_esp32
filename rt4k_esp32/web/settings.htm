Settings

Disable wifi next bootClick this button to stop the ESP32 from connecting to wifi on the next boot, letting you use the SD card at high speed with Windows without interruption. This only applies temporarily to the next boot, to restore wifi, reboot the ESP32 again (unplugging/replugging the SD will suffice).
Reboot ESP32Reboots the ESP32.
Delay refreshing wifi.iniTime (in seconds) after the ESP32 boots before it will try to access the SD card to read the wifi.ini file, which will temporarily disconnect the SD card from the host device (RT4K, Windows, etc).
Lock SD until wifi.ini refreshWhen enabled, lock all access to the SD card (WebDAV, primarily) from accessing the SD card until the wifi.ini timer has elapsed. This prevents anything (like Windows randomly refreshing the WebDAV contents) from interrupt the RT4K boot or firmware update.
